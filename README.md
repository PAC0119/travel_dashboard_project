## Project Summary
End-to-end data analytics project covering India's outbound 
travel market and B2B travel agent performance.

**Tools:** MySQL 8.0 | Power BI Desktop | Excel | Python

## 3 Dashboards Built
| Dashboard | Focus | Key Insight |
|-----------|-------|-------------|
| DB1 India Outbound | Departure trends 2020-2024 | 31.2M departures, 152% recovery 2022 |
| DB2 Asia Market | India-Asia corridor analysis | Japan fastest growing at 42.3% CAGR |
| DB3 B2B Agent | Agent performance + funnel | 28.1% conversion, ₹1,723L FY revenue |

## SQL Skills Demonstrated
- Window Functions: RANK(), LAG(), SUM() OVER()
- Aggregation: GROUP BY, HAVING, CASE statements  
- 12 queries across 6 normalised tables
- Business KPIs: YoY growth, MoM change, conversion rates
